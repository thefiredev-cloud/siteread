# SiteRead

> Reverse-Engineer Any Website

Point at any site — get full tech stack, architecture decisions, and a working rebuild. Not scraping — understanding.

![Screenshot placeholder](https://via.placeholder.com/1200x630/0a0a0f/06b6d4?text=SiteRead)

## Built for Claude Mythos

This landing page was built as part of The Fire Dev's moonshot product suite.

## Tech Stack

- Next.js 14 (App Router, static export)
- React 18
- Three.js via @react-three/fiber + @react-three/drei
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

Outputs to `out/` directory (static export, ready for Netlify/Vercel).

## Features

- **Stack Detection** — Identify every framework, library, CDN, and infrastructure component
- **Architecture Mapping** — Visualize how components connect
- **Decision Analysis** — Understand WHY certain choices were made
- **Instant Rebuild** — Get a working codebase scaffold
