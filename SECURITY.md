# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅        |
| < 1.0   | ❌        |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Email: security@thefiredev.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested mitigations

We will respond within 48 hours and provide a fix timeline within 7 days.

## Responsible Disclosure

We follow coordinated disclosure. Please give us 90 days to patch before public disclosure. We will credit researchers in release notes unless anonymity is requested.
